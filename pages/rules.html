Rules for All Game Types

Standard Pick-Every-Game Pool

  • Pick every game of every round.
  • Slections open once games are set for a round of the tournament.
  • Player has until scheduled start of the first game of the day to submit selections for every game played that day.
  • If no picks are submitted, the player gets no points for any game played that day.
  • First round games are worth 10 points each. Second round 20 points each, and so on, with the points value per game doubling each round.
